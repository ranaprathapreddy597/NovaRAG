# 🐨 Quokka AI
**A High-Performance Hybrid RAG System for Intelligent Document Interaction**

Quokka AI is an advanced Large Language Model (LLM) interface designed to bridge the gap between static AI responses and real-time data retrieval. By leveraging **Retrieval-Augmented Generation (RAG)** and **Llama-3**, Quokka allows users to interact with their own data through a seamless, responsive UI.

---

## 🌟 Key Features
* **Hybrid RAG Architecture:** Seamlessly switches between general knowledge and specific document context (PDF/Text).
* **Llama-3 Integration:** Utilizes high-performance inference for natural, context-aware conversations.
* **Custom VS Code Frontend:** A sleek, user-centric interface designed for low latency.
* **Real-time Web Search:** Integrated API calls for fetching up-to-date information beyond the model's training data.
* **Colab-Optimized Backend:** Scalable backend logic designed to leverage T4 GPU acceleration.

## 🛠️ Tech Stack
* **LLM:** Meta Llama-3-8B
* **Backend:** Python (Google Colab / FastAPI)
* **Frontend:** HTML5, CSS3, JavaScript (Developed in VS Code)
* **Frameworks:** Gradio / LangChain
* **Environment:** Hugging Face Hub / Google Cloud Ecosystem

---

## 🏗️ System Architecture
Quokka AI operates on a **Decoupled Architecture**:
1.  **Inference Layer:** Hosted on Colab, handling heavy GPU computations and vector embeddings.
2.  **Client Layer:** A lightweight, responsive frontend that communicates via secure API endpoints.

2. **Frontend:** Open `frontend/index.html` in your browser.
3. **Connect:** Enter the public URL generated by the backend into the frontend configuration.
